#### D0012 — формула не финализирована
- Это "Шуруп мал. потайной", присутствует в 100% проектов
- Появляется ДВАЖДЫ в каждом проекте
- Первое значение = `ceil(n_fix / 2)` = `D0010 / 2` ✓ (верифицировано)
- Второе значение = `n_total` для большинства прямых проектов
  - Но `В25-5-4372` (sec=2, n=5): второе=6 ≠ n_total=5
  - Для угловых проектов второе значение намного больше (×2-3)
- **Задача:** открыть 3-4 проекта разных систем, посмотреть что вокруг D0012 в PDF
